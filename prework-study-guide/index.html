Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The HTML is the foundation of the code.
  • HTML stands for Hypertext Markup Language.
  • HTML can also be used to add meta information to a webpage.
  • Meta is information about the webpage, which isn't usually shown by web browsers.

CSS

  • CSS is the can be described as the style of the website.
  • CSS stands for cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed.
  • CSS saves a lot of work, it can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS.

Git

  • Git status: checks what branch we are currently on.
  • git check out -b branch-name: creates a new branch and switches to it.
  • Git provides workflow, making it possible for programmers to collaborate on software developement.

JavaScript

  • JavaScript is added to the HTML to make the wesite interactive.
